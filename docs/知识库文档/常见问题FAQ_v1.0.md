# 能耗智能管理优化平台 — 常见问题 FAQ

> 版本：v1.0 | 更新日期：2026-06-21

---

## 一、登录与账号

### Q1: 如何登录平台？

打开浏览器访问平台地址（开发环境：`http://localhost:5173`），输入用户名和密码后点击「登录」按钮。

### Q2: 默认的测试账号是什么？

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | admin123 | 管理员 |

登录页面底部有「测试账号」区域，点击可自动填充。

### Q3: 登录失败怎么办？

| 错误提示 | 解决办法 |
|---------|---------|
| "用户名或密码错误" | 检查用户名密码大小写是否正确，注意区分字母大小写 |
| "用户名至少2个字符" | 输入完整的用户名，不要留空 |
| "密码至少4个字符" | 输入完整的密码，不要留空 |

### Q4: 如何退出登录？

点击页面右上角的用户头像或用户名，在下拉菜单中选择「退出登录」。

### Q5: 忘记密码怎么办？

请联系系统管理员重置密码。当前版本暂未提供自助找回密码功能。

## 二、能耗看板

### Q6: 如何查看设备实时功率？

进入「能耗看板」，在「实时功率趋势」图表中可以查看各设备的功率曲线。鼠标悬停在曲线上可以查看具体数值。

### Q7: 看板上的数据多久刷新一次？

页面加载时自动获取最新数据，部分数据（如功率趋势图）在页面加载时一次性获取。您可以通过刷新页面获取最新数据。

### Q8: 能流桑基图怎么看？

桑基图从左到右展示变压器 → 车间 → 设备的功率流向。节点的宽度代表功率的大小，越宽表示功率越大。鼠标悬停在节点或连线上可查看具体的功率值。

### Q9: 想查看某时间段的功率趋势如何处理？

趋势图默认展示当天数据。X 轴为时间，Y 轴为功率值，可以通过图表右上角的图例切换显示/隐藏特定设备。

## 三、设备管理

### Q10: 如何添加新设备？

1. 点击左侧菜单「设备管理」
2. 点击页面右上角「添加设备」按钮
3. 在弹出的表单中填写设备名称、类型、额定功率、所属车间等信息
4. 点击「确认」保存

### Q11: 如何修改设备信息？

1. 在设备列表中，找到目标设备
2. 点击设备行末的「编辑」按钮
3. 修改需要变更的信息
4. 点击「确认」保存

### Q12: 如何删除设备？

1. 在设备列表中，找到目标设备
2. 点击设备行末的「删除」按钮
3. 在确认弹窗中点击「确认」完成删除

**注意**：删除设备后相关联的遥测数据和告警记录仍会保留，但设备将不再产生新数据。

### Q13: 设备状态有哪些？分别代表什么？

| 状态 | 说明 |
|------|------|
| 在线 | 设备正常运行，正在产生能耗数据 |
| 离线 | 设备已停止或无法连接 |
| 待机 | 设备处于待机状态，能耗较低 |
| 告警 | 设备触发了某个告警阈值 |

### Q14: 支持哪些设备类型？

系统支持以下设备类型：空压机、注塑机、冷水机组、照明。

## 四、电价策略

### Q15: 如何配置分时电价？

1. 点击左侧菜单「电价策略」
2. 点击「添加时段」按钮
3. 选择时段类型（高峰/平段/低谷）
4. 设置开始时间和结束时间（精确到分钟）
5. 输入电价（元/kWh）
6. 点击「确认」保存
7. 重复以上步骤覆盖全天 24 小时

### Q16: 电价时段可以重叠吗？

不可以。系统要求全天 24 小时被时段完整覆盖，时段之间不能有空隙或重叠。

### Q17: 如何查看当前处于哪个时段？

进入「电价策略」页面，页面顶部会实时显示当前所处的电价时段和对应的电价。

## 五、告警管理

### Q18: 如何设置告警阈值？

1. 点击左侧菜单「告警管理」
2. 在「阈值配置」区域，点击「添加阈值」
3. 选择目标设备
4. 设置功率上限和下限
5. 选择严重程度（警告/严重/紧急）
6. 点击「确认」保存

### Q19: 告警如何处理？

1. 进入「告警管理」→「告警记录」
2. 找到未处理的告警
3. 点击该告警行末的「处理」按钮
4. 在弹窗中填写处理措施和处理人
5. 点击「确认」完成处理

### Q20: 告警有哪些等级？

| 等级 | 说明 |
|------|------|
| 警告 | 设备偏离正常运行范围，需关注 |
| 严重 | 设备明显超出安全运行范围，需尽快处理 |
| 紧急 | 设备处于危险运行状态，需立即处理 |

### Q21: 告警历史可以筛选吗？

可以。告警记录支持按设备筛选，也可以选择」仅查看未处理」来聚焦待处理的告警。

## 六、智能体对话

### Q22: 智能体对话怎么打开？

点击页面右下角的「💬」悬浮按钮，即可打开对话窗口。

### Q23: 智能体能做什么？

智能体支持三类功能：
- **能耗分析**：分析指定设备或全部设备的能耗数据（如"分析空压机A-1的能耗"）
- **平台使用问答**：回答平台功能使用相关问题（如"如何添加新设备"）
- **通用对话**：进行普通的问答交流

### Q24: 智能体不回复是什么原因？

请检查以下事项：

1. 确认「系统设置」→「AI 管理」中「启用云端智能体」开关已打开
2. 确认 Coze API Key 和 Bot ID 已正确配置
3. 点击「测试连接」验证服务连通性
4. 如以上均正常，可能是网络问题，请稍后重试

如云端不可用，系统会自动切换到本地模式，对话功能仍可用但回复内容为固定提示。

### Q25: AI 对话过程中状态显示什么含义？

| 状态 | 含义 |
|------|------|
| AI 正在思考... | 智能体正在处理您的问题 |
| AI 正在输入... | 智能体正在生成回复内容 |
| 正在分析设备能耗数据... | 智能体正在调用系统分析功能（此时可处理耗时较长的分析任务） |

### Q26: 如何让智能体分析指定设备的能耗？

在对话中直接输入类似以下的问题：
- "分析空压机A-1的能耗"
- "分析所有设备的能耗情况"
- "今天用电情况怎么样"

智能体会自动识别设备名称并调用分析功能。

## 七、调度优化

### Q27: 调度优化需要哪些输入？

执行优化前需要设置以下参数：
- **生产目标**：需要生产的件数
- **截止时间**：生产任务必须完成的时间
- **可用设备**：可以选择参与生产的设备

### Q28: 优化出来的结果怎么看？

优化结果包含三部分：
1. **甘特图**：各设备在不同时间段的启停安排（绿色=运行，灰色=待机）
2. **成本对比**：优化前后的成本对比及预计节省金额
3. **调度明细表**：每台设备的详细启停时间

### Q29: 调度优化和 AI 智能优化有什么区别？

- **调度优化**：使用系统内置的本地算法，基于电价时段和设备能效进行优化
- **AI 智能优化**：调用云端智能体进行更高级的优化分析，包含更多维度的优化建议

## 八、智能报告

### Q30: 如何执行能耗分析？

1. 进入「智能报告」→「能耗分析」选项卡
2. 选择目标设备（不选则分析全部设备）
3. 点击「开始分析」按钮
4. 查看分析结果摘要、异常检测和节能建议

### Q31: 如何查看历史报告？

1. 进入「智能报告」→「历史报告」选项卡
2. 列表展示历史分析记录
3. 可按时间范围筛选报告
4. 点击「查看详情」可查看报告的完整内容

### Q32: 如何创建报告订阅？

1. 进入「智能报告」→「报告订阅」选项卡
2. 点击「新建订阅」
3. 填写订阅名称、选择报告类型（日报/周报/分析报告）
4. 设置执行时间（每天自动执行的时间）
5. 选择通知方式（系统通知/邮件/钉钉）
6. 如有需要，选择目标设备
7. 点击「保存」

### Q33: 邮件和钉钉通知如何配置？

**邮件配置**：选择通知方式为「邮件」后，会显示配置表单，需填写 SMTP 服务器、端口号、发送邮箱、认证密码等信息。

**钉钉配置**：选择通知方式为「钉钉」后，需填写 Webhook URL、可选的安全令牌等。

### Q34: 报告订阅执行后如何查看结果？

- 通知方式为「系统通知」时，报告生成后消息会出现在页面右上角的铃铛图标中
- 通知方式为「邮件」或「钉钉」时，报告会推送到对应渠道
- 也可以在订阅列表中点击「查看报告」查看该订阅的历史报告

## 九、成本分摊

### Q35: 如何查看车间电费成本？

1. 点击左侧菜单「成本分摊」
2. 页面展示各车间的电费汇总（本月/上月）
3. 点击具体车间可查看该车间的电费明细

### Q36: 成本分摊的数据来源是什么？

成本分摊基于各设备的实时遥测数据（功率）和电价策略（峰平谷电价），按车间维度进行统计计算。

## 十、AI 配置管理

### Q37: AI 配置在哪里设置？

点击左侧菜单「AI 管理」（仅管理员可见）。

### Q38: 云端模式和本地模式有什么区别？

| 模式 | 说明 |
|------|------|
| 云端模式 | 通过 Coze 平台的工作流和智能体进行处理，分析质量更高 |
| 本地模式 | 使用系统内置规则引擎处理，功能受限但不依赖外部服务 |

系统在云端不可用时自动降级到本地模式。

### Q39: 如何测试 AI 配置是否正确？

在「AI 管理」页面中，配置好 API Key 和工作流 ID 后，点击「测试连接」按钮。系统会尝试连接 Coze 平台并返回连接结果。

### Q40: 能耗分析功能没有反应是什么原因？

1. 检查「AI 管理」中「能耗分析」功能是否已启用
2. 检查云端模式下 API Key 和工作流 ID 是否配置正确
3. 查看页面是否有黄色警告提示（云端错误信息）
4. 尝试切换到本地模式排查是否为云端问题

## 十一、用户与权限

### Q41: 系统有哪些用户角色？

| 角色 | 说明 |
|------|------|
| admin（管理员） | 拥有所有权限，包括用户管理和 AI 配置 |
| dispatcher（调度员） | 除用户管理和 AI 配置外的全部功能 |
| operator（操作员） | 设备管理、看板查看、告警处理、分析执行 |
| viewer（观察者） | 查看类功能，不能修改 |

### Q42: 如何创建新用户？

1. 点击左侧菜单「用户管理」
2. 点击「添加用户」按钮
3. 输入用户名、密码，选择角色
4. 点击「确认」保存

### Q43: 如何设置用户的细粒度权限？

1. 进入「用户管理」页面
2. 在用户列表中选择目标用户
3. 在下方的权限矩阵区域勾选该用户可访问的功能模块
4. 点击「保存权限」生效

## 十二、其他

### Q44: 开放 API 文档在哪里？

后端服务启动后，访问 `http://localhost:8000/docs` 查看 Swagger API 文档，或访问 `http://localhost:8000/redoc` 查看 ReDoc 文档。

### Q45: 页面显示 500 错误怎么办？

通常是后端服务异常。请检查：
1. 后端服务是否正常运行
2. 查看后端控制台是否有错误日志
3. 尝试重启后端服务

### Q46: 数据不显示或显示异常怎么办？

1. 检查后端服务是否启动
2. 检查前端是否配置正确的 API 代理地址（`vite.config.js` 中的代理目标）
3. 刷新页面重试

### Q47: 如何导出报表？

进入「报表中心」，选择日报/周报/月报查看，支持能耗数据和告警数据的导出功能。
