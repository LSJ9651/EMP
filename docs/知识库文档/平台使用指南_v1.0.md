# 能耗智能管理优化平台 — 使用指南

> 版本：v1.0 | 更新日期：2026-06-21

---

## 第一章 平台概述

### 1.1 平台简介

**能耗智能管理优化平台**是一套面向工业场景的综合性能耗管理系统，通过数据可视化与智能策略，实现工厂能源成本的精细化管理与用能效率提升。

### 1.2 核心功能

| 功能模块 | 说明 |
|---------|------|
| **能耗看板** | 实时功率、今日能耗、CO₂排放、能流桑基图、功率趋势 |
| **设备管理** | 设备 CRUD、能效排行、能耗详情 |
| **电价策略** | 分时电价配置（峰/平/谷）、当前时段电价 |
| **告警管理** | 阈值配置、告警记录、自动评估、知识库建议 |
| **调度优化** | 基于电价时段的设备启停方案优化，预估节费 |
| **智能报告** | 能耗分析、历史报告、报告订阅（系统/邮件/钉钉通知） |
| **AI 对话助手** | 意图识别 + 自然语言交互，支持能耗分析/设备匹配/通用问答 |
| **成本分摊** | 按车间维度的电费统计与明细 |
| **报表中心** | 日报/周报/月报，能耗与告警数据导出 |
| **AI 配置管理** | 可视化配置 Coze API Key、工作流 ID、云端开关 |

### 1.3 用户角色

| 角色 | 说明 | 主要权限 |
|------|------|---------|
| admin（管理员） | 系统管理员 | 全部功能权限，含用户管理、AI 配置 |
| dispatcher（调度员） | 生产调度人员 | 除用户管理、AI 配置外的全部功能 |
| operator（操作员） | 设备操作人员 | 设备管理、看板查看、告警处理、分析执行 |
| viewer（观察者） | 只读用户 | 查看类功能，不能执行修改操作 |

## 第二章 登录与账号管理

### 2.1 登录入口

访问平台 URL（开发环境：`http://localhost:5173`），进入登录页面。

### 2.2 登录操作

1. 在用户名输入框输入您的用户名
2. 在密码输入框输入您的密码
3. 可选：勾选「记住账号」保存用户名到本地
4. 点击「登录」按钮或按 Enter 键提交

### 2.3 默认测试账号

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | admin123 | 管理员 |

> 提示：登录页面底部有「测试账号」区域，点击可自动填充默认账号信息。

### 2.4 登录失败处理

| 错误提示 | 可能原因 | 解决办法 |
|---------|---------|---------|
| "用户名或密码错误" | 账号密码不匹配 | 检查大小写和输入法状态 |
| "用户名至少2个字符" | 用户名过短 | 输入完整的用户名 |
| "密码至少4个字符" | 密码过短 | 输入完整的密码 |

### 2.5 退出登录

1. 点击页面右上角的用户头像或用户名
2. 在弹出的下拉菜单中点击「退出登录」
3. 系统自动跳转回登录页面

### 2.6 用户管理（管理员）

1. 点击左侧菜单「用户管理」
2. 查看用户列表（用户名、显示名、角色、状态）
3. 点击「添加用户」创建新用户（设置用户名、密码、角色）
4. 点击「编辑」修改用户信息或重置密码
5. 权限矩阵区域可设置用户的细粒度功能权限

## 第三章 能耗看板

### 3.1 访问路径

登录后默认进入能耗看板，也可点击左侧菜单「能耗看板」进入。

### 3.2 核心 KPI 指标

看板顶部展示四个关键指标卡片：

| 指标 | 说明 |
|------|------|
| **实时总功率** | 所有设备当前功率总和（kW） |
| **今日总能耗** | 当日累计能耗（kWh） |
| **碳排放估算** | 基于能耗折算的 CO₂ 排放量（kg） |
| **未处理告警** | 当前未处理的告警数量 |

### 3.3 能流桑基图

展示从变压器 → 车间 → 设备的功率流向，直观反映能耗分布。

- 节点宽度表示功率大小
- 鼠标悬停在节点或连接线上可查看具体数值

### 3.4 实时功率趋势图

展示各设备的功率变化曲线。

- X 轴为时间，Y 轴为功率（kW）
- 不同设备以不同颜色区分
- 鼠标悬停可查看具体时间点的功率值
- 图表右上角图例可点击切换显示/隐藏对应设备

### 3.5 最新告警列表

展示最近产生的告警信息，自动滚动更新。

- 显示告警时间、设备名称、告警类型
- 点击可跳转到告警管理页查看详情

### 3.6 AI 智能分析

- 点击「分析设备能耗数据」按钮，可基于当前设备数据调用智能体进行分析
- 分析结果通过弹窗展示异常检测和节能建议

## 第四章 设备管理

### 4.1 访问路径

点击左侧菜单「设备管理」。

### 4.2 设备列表

展示所有设备的详细信息：

| 字段 | 说明 |
|------|------|
| ID | 系统自动生成的设备唯一标识 |
| 设备名称 | 用户自定义的设备名称 |
| 类型 | 空压机 / 注塑机 / 冷水机组 / 照明 |
| 额定功率 | 设备额定功率（单位：kW） |
| 车间 | 设备所属车间（一车间 / 二车间 / 三车间） |
| 效率 | 设备能效百分比 |
| 状态 | 在线 / 离线 / 待机 / 告警 |

### 4.3 添加设备

1. 点击「添加设备」按钮
2. 在弹出的表单中填写设备信息（名称、类型、额定功率、车间等）
3. 点击「确认」保存

### 4.4 编辑设备

1. 在设备列表中，找到目标设备
2. 点击设备行末的「编辑」按钮
3. 修改设备信息后点击「确认」保存

### 4.5 删除设备

1. 在设备列表中，找到目标设备
2. 点击设备行末的「删除」按钮
3. 在确认弹窗中点击「确认」完成删除

### 4.6 能效排行

页面右侧展示设备的能效排行列表，按效率从高到低排序。

### 4.7 能耗详情查看

点击设备行末的操作按钮可查看该设备的详细能耗分析报告。

### 4.8 AI 分析

点击「分析设备能耗数据」可对选中设备或全厂设备执行智能能耗分析，结果包含异常检测和节能建议。

## 第五章 电价策略配置

### 5.1 访问路径

点击左侧菜单「电价策略」。

### 5.2 电价时段

系统支持三个电价时段：

| 时段类型 | 说明 |
|---------|------|
| **高峰** | 用电紧张时段，电价最高 |
| **平段** | 正常用电时段，电价中等 |
| **低谷** | 用电宽松时段，电价最低 |

### 5.3 当前时段显示

页面顶部实时显示当前所处的电价时段和对应的电价。

### 5.4 添加电价时段

1. 点击「添加时段」按钮
2. 选择时段类型（高峰/平段/低谷）
3. 设置开始时间和结束时间（精确到分钟）
4. 输入该时段电价（元/kWh）
5. 点击「确认」保存

### 5.5 编辑电价时段

1. 在电价列表中，找到目标时段
2. 点击「编辑」按钮
3. 修改时间或价格后点击「确认」保存

### 5.6 时段覆盖要求

系统要求全天 24 小时被时段完整覆盖，时段之间不能有空隙或重叠。

## 第六章 告警管理

### 6.1 访问路径

点击左侧菜单「告警管理」。

### 6.2 阈值配置

#### 6.2.1 查看阈值

阈值配置列表展示所有设备的功率上下限设置。

#### 6.2.2 添加阈值

1. 点击「添加阈值」按钮
2. 选择目标设备
3. 设置功率上限（kW）和功率下限（kW）
4. 选择严重程度（警告/严重/紧急）
5. 点击「确认」保存

#### 6.2.3 编辑/删除阈值

- 点击阈值行末的「编辑」按钮修改设置
- 点击阈值行末的「删除」按钮移除阈值

### 6.3 告警记录

告警记录列表展示以下信息：

| 字段 | 说明 |
|------|------|
| 告警时间 | 告警触发时间 |
| 设备名称 | 触发告警的设备 |
| 参数类型 | 触发参数（功率） |
| 当前值 | 触发时的实际值 |
| 阈值 | 设置的阈值范围 |
| 严重程度 | 警告 / 严重 / 紧急 |
| 状态 | 未处理 / 已处理 |

### 6.4 筛选功能

- 支持按设备筛选告警记录
- 支持仅查看未处理的告警
- 点击「刷新」按钮获取最新告警数据

### 6.5 处理告警

1. 在告警记录列表中，找到未处理的告警
2. 点击「处理」按钮
3. 在弹窗中填写处理措施和处理人
4. 点击「确认」完成处理

### 6.6 告警统计

页面展示告警统计概览：总告警数、未处理数、今日新增数。

### 6.7 处理建议

系统根据告警类型提供知识库建议，辅助操作人员快速定位和解决问题。

## 第七章 智能体对话

### 7.1 访问方式

点击页面右下角的悬浮按钮「💬」，打开对话窗口。

### 7.2 基本操作

1. 在输入框中输入您的问题
2. 按 Enter 键发送消息
3. 智能体会实时流式返回回复内容

### 7.3 智能体能力

#### 7.3.1 能耗分析

当您询问设备能耗相关问题时（如"分析空压机A-1的能耗"、"今天用电情况怎么样"），智能体会：

1. 识别您要分析的设备
2. 自动调用系统能耗分析功能获取数据
3. 生成详细的分析报告（含异常检测和节能建议）

#### 7.3.2 平台使用问答

智能体可以回答关于平台功能使用的各类问题，如操作步骤、功能说明等。

#### 7.3.3 通用对话

智能体具备通用的对话能力，可以进行普通的问答交流。

### 7.4 响应状态

对话过程中，输入框上方会显示当前状态：

| 状态 | 说明 |
|------|------|
| AI 正在思考... | 智能体正在处理您的问题 |
| AI 正在输入... | 智能体正在生成回复内容 |
| 正在分析设备能耗数据... | 智能体正在调用系统分析功能 |

### 7.5 对话历史

打开对话窗口时，系统会自动加载最近的对话历史记录。

### 7.6 查询示例

建议的查询方式：

| 查询类型 | 示例 |
|---------|------|
| 分析指定设备 | "分析空压机A-1的能耗" |
| 分析全部设备 | "分析所有设备的能耗情况" |
| 今日用电查询 | "今天用电情况怎么样" |
| 平台操作咨询 | "如何添加新设备" |

## 第八章 调度优化

### 8.1 访问路径

点击左侧菜单「调度优化」。

### 8.2 输入参数

在执行优化前需要设置以下参数：

| 参数 | 说明 | 示例 |
|------|------|------|
| 生产目标 | 需要生产的件数 | 1000 件 |
| 截止时间 | 生产任务的截止时间 | 18:00 |
| 可用设备 | 可选择参与生产的设备 | 空压机A-1、注塑机A-1 等 |

### 8.3 执行优化

1. 填写生产目标、截止时间
2. 选择可用的生产设备
3. 点击「开始优化」按钮
4. 系统基于电价时段和设备能效进行计算

### 8.4 优化结果解读

优化完成后展示以下内容：

#### 8.4.1 甘特图

以甘特图形式展示各设备在不同时间段的启停安排。

- X 轴为时间线
- Y 轴为设备
- 不同颜色块表示运行/待机状态

#### 8.4.2 成本对比

| 指标 | 说明 |
|------|------|
| 优化前成本 | 按当前运行模式估算的电费 |
| 优化后成本 | 按优化方案运行的电费 |
| 预计节约 | 优化前后的成本差额 |
| 节约比例 | 成本降低百分比 |

#### 8.4.3 调度明细表

展示每台设备的详细调度计划：

| 字段 | 说明 |
|------|------|
| 设备名称 | 参与调度的设备 |
| 开始时间 | 设备启动时间 |
| 结束时间 | 设备停止时间 |
| 动作 | 运行 / 待机 |

### 8.5 AI 智能优化

点击「AI 智能优化」可调用云端智能体进行更高级的优化分析，返回包含更多维度的优化建议。

## 第九章 智能报告

### 9.1 访问路径

点击左侧菜单「智能报告」。

### 9.2 能耗分析

#### 9.2.1 执行分析

1. 在「能耗分析」选项卡中，选择目标设备（默认为全部设备）
2. 点击「开始分析」按钮
3. 系统执行分析并展示结果

#### 9.2.2 分析结果

分析结果包含：

- **摘要**：各设备能耗概况
- **异常检测**：列出检测到的异常设备及详情（设备名称、严重程度、异常描述）
- **节能建议**：系统提供的节能优化建议列表

### 9.3 历史报告

#### 9.3.1 查看历史报告

1. 切换到「历史报告」选项卡
2. 列表展示历史分析记录（报告类型、触发时间）
3. 支持按时间范围筛选

#### 9.3.2 查看报告详情

1. 点击历史报告行末的「查看详情」按钮
2. 弹窗展示报告的完整内容（含异常检测、建议、原始数据）

### 9.4 报告订阅

#### 9.4.1 创建订阅

1. 切换到「报告订阅」选项卡
2. 点击「新建订阅」按钮
3. 填写以下信息：
   - **名称**：订阅的名称
   - **报告类型**：日报 / 周报 / 分析报告
   - **执行时间**：每天自动执行的时间
   - **通知方式**：系统通知 / 邮件 / 钉钉
   - **目标设备**：选择需要分析的设备（可选）
   - **启用状态**：是否立即启用

#### 9.4.2 通知方式配置

**系统通知**：无需额外配置，报告生成后在消息中心查看。

**邮件通知配置**：

| 字段 | 说明 |
|------|------|
| SMTP 服务器 | 邮件服务器地址（如 smtp.example.com） |
| 端口号 | SMTP 端口（25/465/587） |
| 发送邮箱 | 发件人邮箱地址 |
| 认证密码 | SMTP 授权码或密码 |
| 启用 TLS | 是否启用加密传输（推荐） |
| 邮件模板 | 可选，自定义邮件正文 |

**钉钉通知配置**：

| 字段 | 说明 |
|------|------|
| Webhook URL | 钉钉机器人 Webhook 地址 |
| 访问令牌 | 钉钉机器人安全令牌（可选） |
| 消息模板 | 可选，自定义消息内容 |
| 通知频率 | 即时发送 / 批量汇总 / 每日摘要 |

#### 9.4.3 管理订阅

- **编辑**：点击「编辑」修改订阅配置
- **立即执行**：点击「立即执行」手动触发一次报告生成
- **查看报告**：查看该订阅产生的历史报告
- **删除**：点击「删除」移除订阅
- **启用/禁用**：切换订阅的启用状态

#### 9.4.4 消息中心

系统通知产生的消息可在页面右上角的铃铛图标中查看。

## 第十章 AI 配置管理

### 10.1 访问路径

点击左侧菜单「AI 管理」（仅管理员可见）。

### 10.2 总开关

「启用云端智能体」开关控制是否启用所有云端 AI 能力：

- **开启**：系统尝试连接 Coze 云端服务进行处理
- **关闭**：系统使用本地规则引擎降级处理

### 10.3 API 配置

| 配置项 | 说明 |
|--------|------|
| Coze API Key | Coze 平台的 API 访问令牌 |
| API 端点 | Coze API 地址（默认 `https://api.coze.cn`） |

### 10.4 工作流配置

| 配置项 | 说明 |
|--------|------|
| 能耗分析 Bot ID | 用于能耗分析的 Coze 智能体 ID |
| 调度优化 Workflow ID | 用于调度优化的 Coze 工作流 ID |
| AI 对话 Bot ID | 智能对话助手的 Coze Bot ID |

### 10.5 超时设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 工作流超时 | 调用 Coze 工作流的最长等待时间 | 120 秒 |
| 对话超时 | AI 对话响应的最长等待时间 | 30 秒 |

### 10.6 连接测试

配置完成后，点击「测试连接」按钮验证与 Coze 平台的连通性。

### 10.7 云端/本地模式判定规则

系统按以下顺序判断使用云端还是本地模式：

1. 检查 `启用云端智能体` 总开关是否开启
2. 检查各服务的子开关是否开启
3. 检查 API Key 是否已配置
4. 检查 Workflow ID / Bot ID 是否已配置

任一条件不满足，对应服务自动降级为本地规则引擎。

### 10.8 各服务独立控制

| 服务 | 独立开关 | 云端模式 | 本地模式 |
|------|---------|---------|---------|
| 能耗分析 | 有 | 调用 Coze 工作流 | 基于限额检测+峰谷电价策略生成分析结果 |
