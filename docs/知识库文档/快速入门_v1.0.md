# 能耗智能管理优化平台 — 快速入门

> 5 分钟快速上手 | 版本：v1.0 | 更新日期：2026-06-21

---

## 第一步：登录系统

1. 打开浏览器访问平台地址（开发环境：`http://localhost:5173`）
2. 输入测试账号：用户名 `admin`，密码 `admin123`
3. 点击「登录」按钮或按 Enter 键
4. 成功登录后自动跳转到「能耗看板」

> 提示：登录页面底部有「测试账号」区域，点击可自动填充账号信息。

---

## 第二步：浏览能耗看板

登录后首先看到的是能耗看板，包含：

| 区域 | 查看内容 |
|------|---------|
| **顶部 KPI** | 实时总功率、今日总能耗、碳排放、未处理告警数 |
| **能流桑基图** | 变压器→车间→设备的功率流向 |
| **功率趋势图** | 各设备功率随时间变化曲线 |
| **最新告警** | 滚动展示最新的告警信息 |

无需任何操作，所有数据自动加载。

---

## 第三步：查看设备列表

1. 点击左侧菜单「设备管理」
2. 查看所有设备的列表（名称、类型、额定功率、车间、状态）
3. 点击某设备行末的「编辑」可修改设备信息

---

## 第四步：配置电价策略

1. 点击左侧菜单「电价策略」
2. 页面顶部显示当前所处时段及电价
3. 若时段列表为空，点击「添加时段」配置峰/平/谷三段的起止时间和电价
4. 确保全天 24 小时被完整覆盖

---

## 第五步：执行一次能耗分析

1. 点击左侧菜单「智能报告」
2. 在「能耗分析」选项卡中，选择目标设备（或留空全部设备）
3. 点击「开始分析」按钮
4. 查看分析结果：设备能耗概况、异常检测、节能建议

---

## 第六步：尝试 AI 对话

1. 点击页面右下角的「💬」悬浮按钮
2. 在输入框中输入问题，例如：`"分析空压机A-1的能耗"`
3. 按 Enter 键发送
4. 智能体会实时返回分析结果

其他可以尝试的提问：
- "今天用电情况怎么样"
- "如何添加新设备"
- "分析所有设备的能耗情况"

---

## 第七步：创建报告订阅

1. 进入「智能报告」→「报告订阅」选项卡
2. 点击「新建订阅」
3. 填写名称、选择报告类型（日报/周报）、设置执行时间
4. 选择通知方式（系统通知/邮件/钉钉）
5. 点击「保存」
6. 系统将在指定时间自动生成报告并通知您

## 遇到问题？

- **登录失败**：检查用户名密码是否正确
- **页面空白**：检查后端服务是否已启动（`uvicorn main:app --reload`）
- **AI 不回复**：检查「AI 管理」页面中云端开关和 API 配置
- **分析结果为空**：确认系统中已添加至少一台设备并有能耗数据
