## 扣子智能体数据格式

### 1. 对话智能体 (Chat Bot) — `coze.chat.create_and_poll()`

**传入**（通过 `Message.build_user_question_text()` 构建）：

```json
{
  "role": "user",
  "content": "用户输入的文本消息",
  "content_type": "text"
}
```

**接收**（从 `chat_result.messages` 提取）：

```json
{
  "role": "assistant",
  "content": "智能体回复的纯文本"
}
```

### 2. 能耗分析工作流 — `coze.workflows.runs.create()`

**传入**（[`_build_analysis_params()`](file:///c:/Users/LSJ/Desktop/能耗管理优化系统/energy-optimizer-platform/backend/services/agent_adapter.py#L91-L99)）：

```json
{

  "device_name": "空压机 A-1",
  "rated_power": 75.0,
  "start_time": "2026-06-19T00:00:00",
  "end_time": "2026-06-19T23:59:59",

```

若返回 JSON 字符串，系统自动 `json.loads()` 反序列化。

### 3. 调度优化工作流 — `coze.workflows.runs.create()`

**传入**（[`_build_optimize_payload()`](file:///c:/Users/LSJ/Desktop/能耗管理优化系统/energy-optimizer-platform/backend/services/agent_adapter.py#L151-L155)）：

```json
{
  "production_goal": 1000,
  "deadline": "2026-06-19T23:59:59",
  "available_devices": [
    {"id": 1, "name": "空压机A-1", "rated_power": 75.0, "run_power": 63.75, "idle_power": 11.25}
  ],
  "tariff_policy": [
    {"period": "高峰", "start": "08:00", "end": "11:00", "price": 1.20},
    {"period": "平段", "start": "11:00", "end": "18:00", "price": 0.80},
    {"period": "高峰", "start": "18:00", "end": "22:00", "price": 1.20},
    {"period": "低谷", "start": "22:00", "end": "06:00", "price": 0.30}
  ]
}
```

**接收**：

```json
{
  "schedule": [
    {"device_id": 1, "action": "run", "start_time": "22:00", "end_time": "06:00", "price_per_kwh": 0.30},
    {"device_id": 1, "action": "run", "start_time": "06:00", "end_time": "08:00", "price_per_kwh": 0.80}

```

若 `estimated_cost_saved` 为 0，[`_estimate_savings_from_schedule()`](file:///c:/Users/LSJ/Desktop/能耗管理优化系统/energy-optimizer-platform/backend/services/agent_adapter.py#L345-L392) 自动重算。

### 通用处理逻辑

- **调用方式**：工作流用 `coze.workflows.runs.create()`，对话用 `coze.chat.create_and_poll()`
- **返回值包装**：统一返回 `{"data": workflow_run.data, "execute_id": workflow_run.execute_id}`
- **异常容错**：`CozeAPIError` 时记录 `code`/`msg`/`debug_url`，上层自动降级本地算法
- **字符串反序列化**：`data` 为字符串时自动 `json.loads()`
